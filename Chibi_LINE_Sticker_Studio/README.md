# Q 版 LINE 貼圖提示詞產生器 & 成品切圖器

這是一個純靜態網頁工具，可直接用瀏覽器開啟，也可以部署到 GitHub Pages、Netlify、Vercel 等任何靜態託管平台。

功能包含：
- 4x4 / 4x6 版型的提示詞產生
- 4x4 / 4x6 貼圖總表切圖、去背、加白邊
- 自動輸出 LINE 貼圖尺寸與縮圖 ZIP

## 使用方式

### 直接開啟
1. 下載或 Clone 專案
2. 用瀏覽器直接開啟 [index.html](index.html)

### 可選：用靜態伺服器
若需要本地端伺服器，使用任何靜態伺服器即可（例如 VS Code 的 Live Server）。

## 部署到靜態平台
將整個資料夾上傳到 GitHub Pages / Netlify / Vercel 即可，無需建構流程。
